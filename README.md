# Банка ДСК покупки на Кредит — Shopify снипет

**Версия:** 1.2.2

Shopify снипет, който добавя възможност за закупуване на кредит на стоките в **продуктовата страница** и в **страницата за количката**. Интеграцията е с **Банка ДСК**.

## Начини на инсталиране

Инсталацията може да се извърши по два начина:

1. **От собственика/администратора на магазина** — следвайки инструкциите по-долу.
2. **От поддържащата фирма** — чрез **Shopify Collaborator Access** (вижте следващия раздел).

След получаване на достъп, стъпките по инсталацията са **еднакви** и за двата случая.

---

## Инсталация от поддържащата фирма (Shopify Collaborator Access)

Ако инсталацията ще я извърши поддържащата фирма на софтуера, администраторът на магазина трябва първо да предостави достъп.

### Стъпка 1: Администраторът генерира код за достъп

1. Влезте в **Shopify Admin** на магазина.
2. Отидете на **Settings** (Настройки) → **Users** (или **Users and permissions**, в зависимост от версията на админа).
3. Отворете раздела **Security** (Сигурност).
4. В секцията **Collaborators** кликнете върху **4-цифрения код**, за да го копирате.
5. Изпратете на поддържащата фирма:
   - **URL на магазина** (напр. `вашият-магазин.myshopify.com`)
   - **4-цифрения collaborator код**
   - **Email адреса**, предоставен от поддържащата фирма на софтуера (използва се при изпращане на заявката за достъп)

> Ако искате да ограничите достъпа само до хора с код: в същия раздел изберете опцията **Only people with a collaborator request code can send a request**.

### Стъпка 2: Поддържащата фирма изпраща заявка за достъп

Поддържащата фирма (Shopify Partner) изпраща заявка от своя **Partner Dashboard**:

1. [partners.shopify.com](https://partners.shopify.com) → **Stores**
2. **Add store** → **Request access to store**
3. Въведете URL на магазина (`*.myshopify.com`) и **4-цифрения код**
4. В **Permissions** изберете поне **Themes** (и други права, ако са необходими)
5. **Request access to store**

### Стъпка 3: Администраторът одобрява заявката

След изпращане на заявката администраторът получава имейл и известие в админа.

1. **Settings** → **Users**
2. Филтрирайте по статус **Requests** (Заявки)
3. Отворете заявката от продавача
4. Прегледайте правата (препоръчително: **Themes** за редакция на темата)
5. **Accept request** (Приеми заявката)

След одобрение поддържащата фирма влиза в магазина през Partner Dashboard и извършва инсталацията по инструкциите по-долу.

## Инсталация на снипета

> Инсталационният архив (`CC_Shopify_DSK_API_v.1.2.2.zip` или актуалната версия) трябва да е наличен при администратора (ако той инсталира) или при поддържащата фирма (ако инсталира през Collaborator Access).

### 1. Отваряне на редактора на кода на темата

1. **Shopify Admin** → **Online Store** → **Themes**
2. При желаната тема: **⋯** (трите точки) → **Edit code** (Редактирай кода)

Вляво се показва дървото с файловете на темата (`assets`, `config`, `snippets`, `sections` и др.).

### 2. Конфигурация в `settings_schema.json`

1. В редактора отворете **Config** → `settings_schema.json`
2. Намерете **предпоследния** обект в масива (редът, който завършва с `}`)
3. Добавете **запетая** след затварящата `}` на този обект
4. Между предпоследния обект и последния ред `]` поставете JSON блока за настройките **„Банка ДСК покупки на Кредит“**

Пълният блок за копиране е в файла [`install-settings.txt`](install-settings.txt).

**Важно:** Файлът трябва да остане валиден JSON. След редакция натиснете **Save**.

### 3. Копиране на файловете

Създайте/качете файловете на съответните места в темата:

#### Assets (`assets/`)

- `dsk-cart-button.css`
- `dsk-cart-button.js`
- `dsk-product-button.css`
- `dsk-product-button.js`
- `dsk-logo-color.svg`

#### Snippets (`snippets/`)

- `dsk-cart-button.liquid`
- `dsk-product-button.liquid`

За всеки нов файл: **Add a new snippet** / **Add a new asset** → създайте файла → копирайте съдържанието → **Save**.

### 4. Бутон в продуктовата страница

Добавете извикването на снипета на подходящо място в шаблона на продуктовата страница:

```liquid
{% render 'dsk-product-button', product: product %}
```

**Пример:** В темата Dawn и производни теми често мястото е `snippets/buy-buttons.liquid` — точното място зависи от използваната тема и е на усмотрение на администратора (обикновено до или под бутоните „Добави в количката“ / „Купи сега“).

**Навигация в редактора:** **Edit code** → **Snippets** → отворете съответния файл → поставете реда → **Save**.

#### По желание: CSS подравняване (продуктова страница)

В **Online Store** → **Themes** → **Customize** (Персонализирай) → **Theme settings** → **Custom CSS** (ако темата го поддържа):

```css
.dskapi-slot {
  margin-top: 1rem;
}
```

### 5. Бутон в страницата за количка

```liquid
{% render 'dsk-cart-button' %}
```

**Пример:** В някои теми подходящ файл е `sections/main-cart-footer.liquid` — отново зависи от темата.

**Навигация:** **Edit code** → **Sections** → `main-cart-footer.liquid` (или друг подходящ файл) → **Save**.

#### По желание: CSS подравняване (количка)

```css

```

### 6. Настройки на темата

1. **Online Store** → **Themes** → **Customize**
2. **Theme settings** (Настройки на темата) → **Банка ДСК покупки на Кредит**
3. Попълнете настройките според договореното с Банка ДСК покупки на Кредит:
   - Уникален идентификатор на магазина
   - Параметри за визуализация на бутона
4. **Save**

### 7. Тест

1. Запазете всички промени в темата
2. Презаредете продуктова страница и страница за количка
3. Проверете показването на бутоните и изпращането на тестова заявка

## Бележки

- Снипетът е **зависим от темата** — файловете `buy-buttons.liquid`, `main-cart-footer.liquid` и др. са специфични за конкретна тема и служат само като ориентир.
- При нужда от допълнително подравняване използвайте **Custom CSS** в персонализатора на темата или стилове в родителската секция.
- За актуализации на версията проверявайте номера на версията в началото на този README.
